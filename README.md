# Vuedoo

The goal of the project is to completely encapsulate the basic boilerplate required for creating even basic Vue project when using Webpack.

## Usage

```
{
  "scripts": {
    "build": "vuedoo build",
    "monitor": "vuedoo monitor",
    "start": "vuedoo"
  },
  "dependencies": {
    "vuedoo": "latest"
  }
}
```
